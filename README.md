# 📌 Project Overview

The Retail ETL & Data Warehouse Analytics Dashboard is an end-to-end data engineering project designed to process retail sales data efficiently. The project follows the ETL (Extract, Transform, Load) process to clean, validate, and store data in a MySQL Data Warehouse. Finally, an interactive Streamlit dashboard is built to visualize business insights through charts, KPIs, and reports.

---

# 🎯 Project Objectives

- Build a complete ETL pipeline using Python.
- Clean and transform raw retail sales data.
- Load processed data into a MySQL Data Warehouse.
- Create an interactive Streamlit dashboard.
- Generate meaningful business insights using data visualization.
- Demonstrate a real-world data engineering workflow.

# 📂 ETL Workflow

### 1️⃣ Extract
- Load the raw retail dataset.
- Read data using Pandas.

### 2️⃣ Validate
- Check missing values.
- Identify duplicate records.
- Validate data quality.

### 3️⃣ Transform
- Clean the dataset.
- Standardize column names.
- Prepare data for analysis.

### 4️⃣ Load
- Load processed data into MySQL Data Warehouse.
- Store structured data for reporting.
